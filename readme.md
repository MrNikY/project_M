# First comand for git
## first comand

```
git init
git add README.md   or  git add *
git commit -m "AddToPrevious"
git branch -M main
git remote add origin git@github.com:MrNikY/project_M.git
git push -u origin main
```

## after

```
git add .
git commit -m 'AddToPrevious'
git push -u origin main
```
